# PRiSM_MusicGestureRecognition
 
PRiSM Music Gesture Recognition is a software tool for real-time recognition of musical gestures based on audio input. It utilizes machine learning techniques to classify and interpret musical gestures, enabling applications in interactive music performance, composition, and more.

<span style="color:red">Note: This software is in beta version and may contain bugs. Use with caution and at your own risk.</span>
